# Marketing Campaign Analysis

## Objective
Evaluate campaign effectiveness and customer spending behavior to improve marketing ROI.

## KPIs
- Total Customers
- Average Income
- Total Spending
- Campaign Acceptance

## Tools Used
- Excel
- Business Analytics
- Data Visualization

## Key Insights
- Customer spending differs across product categories.
- Certain campaigns achieved higher response rates.
- High-value customers can be targeted for future campaigns.

## Recommendations
- Focus on high-performing campaigns.
- Improve underperforming campaigns using testing and segmentation.
